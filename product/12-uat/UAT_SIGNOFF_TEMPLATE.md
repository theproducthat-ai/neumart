## Next Step After Sign-off

- **Passed:** Proceed to Release Planning. Link this UAT file in the release plan.
- **Failed:** Return to development. Create new dev task. Re-run QA and UAT.
- **Conditional Pass:** Fix listed conditions. Confirm completion. Proceed to release.
