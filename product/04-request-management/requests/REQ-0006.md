## Request Summary

Allow customers to filter products on the Product Listing page by allergen (exclude products containing a specific allergen) and by dietary tag (show only Vegan, Gluten-free, Jain-friendly products, etc.).

## Scope (When Reopened)

- Filter panel or chips on SCR-CUS-0001 for dietary tags (Vegan, Gluten-free, Jain-friendly, Organic, etc.)
- Exclude-allergen filter (e.g. "Exclude: Peanuts, Gluten")
- Filters should work on the structured `containsAllergens` and `dietaryTags` array fields added by REQ-0005
- No schema changes needed — REQ-0005 schema is already designed for this

## Why Parked

REQ-0005 adds the schema fields and data capture. Without allergen data on products, filters return empty or misleading results. This request must wait until REQ-0005 is shipped and the product catalogue has been enriched with allergen data.
