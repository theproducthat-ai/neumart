## Request Description

Build a Delivery Module for Nuemart. The user has requested a delivery capability but has not yet specified which sub-modules are in scope (assignment, status tracking, customer visibility, proof of delivery, etc.). Full scope to be determined during grilling.

---

## Why This Matters

Nuemart currently has no delivery tracking or fulfilment management. All delivery is handled manually with no system visibility. A delivery module would improve operational efficiency and customer experience post-order.
