## Request Description

The favourite/heart icon currently sits in the bottom action row of the product card, alongside the "Add to cart" button. This clutters the action row and breaks a common e-commerce pattern.

The request is to move the heart icon to an absolutely positioned overlay on the product image — top-right corner — with a small white or semi-transparent circular background for visibility. The bottom action row should then contain only the "Add to cart" button.

## Technical Notes

- Heart icon becomes an absolutely positioned overlay inside the product image container
- Positioned at top-right corner (e.g. `top-2 right-2`)
- Small circular background: white or semi-transparent (e.g. `bg-white/80 rounded-full p-1.5`)
- The bottom action row no longer renders the heart icon — only "Add to cart" remains
- No schema change, no business logic change, no API change
- Primary file: `neumart/components/products/product-card.tsx`
