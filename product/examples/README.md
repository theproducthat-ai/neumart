## How to Read These Examples

Each example shows:
1. **The intake** — what was received and from whom
2. **Classification** — which lane was assigned and why
3. **Required artifacts** — what the team needed to produce
4. **Object chain** — how the objects link together
5. **Outcome** — how it was delivered

These are illustrative — the object IDs, dates, and content are realistic but not necessarily actual records.
