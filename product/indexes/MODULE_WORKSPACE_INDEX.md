## How to Use

1. Find the module you want to explore by name or ID.
2. Open its workspace file linked in the table above.
3. The workspace shows all linked features, bugs, PRDs, stories, roadmap items, and open work for that module.
4. Follow links to `product/objects/` for source-of-truth object details.
