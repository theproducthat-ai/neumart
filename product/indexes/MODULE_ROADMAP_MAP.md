## Purpose

Shows what is on the roadmap for each module — what is planned, what is in progress, and what has recently shipped. Use this to answer:
- "Show me all roadmap items under Admin Console"
- "What is coming for Delivery Management next quarter?"
- "Which module has the most roadmap work this sprint?"

---

## How to Read This Map

- **This sprint**: features in active development or QA this sprint
- **Next sprint**: planned and scheduled features
- **This quarter**: all features committed for the quarter
- **Backlog**: approved features not yet scheduled
