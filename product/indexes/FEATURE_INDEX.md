## Feature Status Definitions

| Status | Meaning |
|---|---|
| `planned` | On roadmap, not yet started |
| `in-design` | Design in progress |
| `in-development` | Engineering in progress |
| `in-qa` | In QA/UAT |
| `released` | Live in production |
| `deprecated` | No longer supported |
| `post-release` | Live and under post-release measurement |

## Feature ID Format

`FEAT-[MODULE]-[SCREEN]-[NNN]`

Examples:
- `FEAT-COM-LIST-001` — Commerce, Product Listing page, feature 001
- `FEAT-ADM-ORD-003` — Admin, Orders page, feature 003
- `FEAT-DEL-MAP-001` — Delivery, Map screen, feature 001
