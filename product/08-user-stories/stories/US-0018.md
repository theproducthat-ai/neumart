## User Story

```
As a customer,
I want to see a compact allergen indicator on the product card,
So that I can quickly identify products I should investigate further before adding to my cart.
```

---

## Acceptance Criteria

- [ ] **AC-1 (containsAllergens badge):**
  Given a product has `containsAllergens` with one or more values,
  When a customer views the product listing,
  Then the product card shows a badge reading **"Contains allergens"**

- [ ] **AC-2 (mayContainAllergens badge — no containsAllergens):**
  Given a product has `mayContainAllergens` with one or more values AND `containsAllergens` is absent or empty,
  When a customer views the product listing,
  Then the product card shows a badge reading **"May contain allergens"**

- [ ] **AC-3 (containsAllergens takes precedence):**
  Given a product has both `containsAllergens` and `mayContainAllergens` set,
  When a customer views the product listing,
  Then the product card shows only **"Contains allergens"** — not both badges

- [ ] **AC-4 (No badge when no data):**
  Given a product has no allergen data,
  When a customer views the product listing,
  Then no allergen badge appears on the product card

- [ ] **AC-5 (Badge is not interactive):**
  Given a product card shows an allergen badge,
  When a customer clicks or taps on the badge,
  Then nothing happens — no tooltip, modal, or link is triggered

- [ ] **AC-6 (Empty array treated as absent):**
  Given a product has `containsAllergens: []` (empty array),
  When a customer views the listing,
  Then no badge is shown — empty array treated same as absent

---

## Business Rules

| # | Rule |
|---|---|
| BR-1 | `containsAllergens?.length > 0` → badge reads "Contains allergens" |
| BR-2 | `mayContainAllergens?.length > 0` AND `containsAllergens` absent/empty → badge reads "May contain allergens" |
| BR-3 | Both set → "Contains allergens" badge only |
| BR-4 | Neither set (or both empty arrays) → no badge |
| BR-5 | Badge is display-only. No click handler, no tooltip, no modal, no link in MVP |
| BR-6 | "No allergens" or any affirmative safety claim must never be displayed on the card |
| BR-7 | Dietary tags alone do not trigger an allergen badge |
