# Requirement

**ID:** `REQ-SPEC-COM-CART-COUPON-002` | **Priority:** Must Have
**PRD:** PRD-COM-CART-COUPON-V1 | **Feature:** FEATURE-COM-CART-COUPON

## Statement

A user with the admin role can view a paginated list of all coupons at `/admin/coupons`. Each row displays: code, discount value (%), max discount cap (₹), minimum cart value (₹), valid from / to, status (Active/Inactive), usage count (total redeemed / limit).
