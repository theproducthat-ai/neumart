# Figma Links

Stores Figma link objects — references to Figma files, pages, frames, and components used across the product. Centralising these allows any product object to reference a design with full context.

## ID Format
`FIG-NNN`

## Linked Objects
- screens: `product/objects/screens/`
- features: `product/objects/features/`
- PRDs: `product/objects/prds/`
- roadmap items: `product/objects/roadmap-items/`

## Template
`product/os/templates/FIGMA_LINK_TEMPLATE.md`

## Policy
`product/os/policies/FIGMA_REFERENCE_RULES.md`
