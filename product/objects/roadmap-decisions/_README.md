# Roadmap Decisions

Records the outcome of roadmap discussions — which option was chosen, who decided, why, and what the next step is.

## ID Format
`RMD-NNN`

## Lifecycle
`pending` → `decided` → `implemented` | `reversed`

## Linked Objects
- parent roadmap item: `product/objects/roadmap-items/`
- selected option: `product/objects/roadmap-options/`
- resulting feature or request: `product/objects/features/` or `product/objects/requests/`

## Template
`product/os/templates/ROADMAP_DECISION_TEMPLATE.md`
