# Reference Materials

Stores links and references to external context: PDFs, spreadsheets, meeting recordings, research reports, competitor analysis, client emails, regulatory documents.

## ID Format
`REF-NNN`

## Linked Objects
- roadmap items: `product/objects/roadmap-items/`
- PRDs: `product/objects/prds/`
- features: `product/objects/features/`
- discovery: `product/objects/discovery/`

## Template
`product/os/templates/REFERENCE_MATERIAL_TEMPLATE.md`

## Note
Reference materials do not duplicate source files. They store pointers, descriptions, and context for why the material is relevant.
