# Screenshots

Stores screenshot reference objects. Screenshots are used to capture the current state of a screen, a design, a bug, or a competitive example. The image file itself is stored in a shared drive or repo assets folder; this object stores the pointer and context.

## ID Format
`SCT-NNN`

## Linked Objects
- screens: `product/objects/screens/`
- bugs: `product/objects/bugs/`
- roadmap items: `product/objects/roadmap-items/`
- competitive research: `product/objects/research-notes/`

## Template
`product/os/templates/SCREENSHOT_REFERENCE_TEMPLATE.md`
