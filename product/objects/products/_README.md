# Products

**Object type**: `product`  
**Owner**: Product Lead  
**Note**: This folder is for product-level objects — used when Neumart expands to multiple distinct products (e.g., a B2B product, a white-label product, or a separate mobile app). For a single-product company, this may remain empty.

## Current Product

- `Neumart` — the core grocery ecommerce platform

## When to Use

- When a second product is launched
- When a white-label or B2B variant is built
- When tracking product-level strategy and health separately from individual features

## Format

`PROD-[NAME].md` with fields:
`id`, `name`, `type`, `target_segment`, `status`, `modules_in_scope`, `owner`
