# Wireframes

Stores wireframe reference objects. Wireframes may be rough sketches, low-fidelity Figma frames, Balsamiq mockups, or annotated screenshots. This folder stores the pointer and context; the actual file lives in Figma, a shared drive, or repo assets.

## ID Format
`WFR-NNN`

## Linked Objects
- screens: `product/objects/screens/`
- features: `product/objects/features/`
- PRDs: `product/objects/prds/`
- flow diagrams: `product/objects/flow-diagrams/`

## Template
`product/os/templates/WIREFRAME_REFERENCE_TEMPLATE.md`
