# Attachments

Stores attachment references for product objects — links to files in Google Drive, Notion, Confluence, email threads, Figma files, spreadsheets, or uploaded documents. Attachments are pointers, not the files themselves.

## ID Format
`ATT-NNN`

## Linked Objects
- Any product object can have linked attachments.
- Common parents: PRDs, features, roadmap items, discovery, incidents.

## Template
`product/os/templates/ATTACHMENT_REFERENCE_TEMPLATE.md`
