# Roadmap Items

Stores all items under active roadmap consideration. A roadmap item is a potential future capability — it may not yet be a feature or PRD. It sits between idea and commitment.

## ID Format
`RMI-NNN`

## Lifecycle
`idea` → `under-discussion` → `approved` → `converted` | `rejected` | `deferred`

## Linked Objects
- converted features: `product/objects/features/`
- converted requests: `product/objects/requests/`
- options: `product/objects/roadmap-options/`
- decisions: `product/objects/roadmap-decisions/`
- reference materials: `product/objects/reference-materials/`
