# Flow Diagrams

Stores references to user flows, process flows, system flows, data flows, and decision trees. These may be Figma files, Miro boards, FigJam diagrams, or text-based Mermaid diagrams stored in the repo.

## ID Format
`FLW-NNN`

## Linked Objects
- features: `product/objects/features/`
- PRDs: `product/objects/prds/`
- roadmap items: `product/objects/roadmap-items/`
- screens: `product/objects/screens/`

## Template
`product/os/templates/FLOW_DIAGRAM_REFERENCE_TEMPLATE.md`
