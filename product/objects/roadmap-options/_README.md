# Roadmap Options

Stores alternative approaches or options for a roadmap item under discussion. When multiple ways to solve a problem exist, each option is documented here for comparison before a decision is made.

## ID Format
`RMO-NNN`

## Lifecycle
`draft` → `under-review` → `selected` | `rejected`

## Linked Objects
- parent roadmap item: `product/objects/roadmap-items/`
- decision: `product/objects/roadmap-decisions/`

## Template
`product/os/templates/ROADMAP_OPTION_TEMPLATE.md`
