# Parking Lot

Holds requests and ideas that are intentionally set aside — not ready to evaluate, not enough context, or deliberately deferred without a target timeline. Items here are reviewed periodically and either promoted to backlog or archived.

## ID Format
`PLT-NNN`

## Lifecycle
`parked` → `backlog` | `archived`

## Linked Objects
- source requests in `product/objects/requests/`
- template: `product/os/templates/PARKING_LOT_ITEM_TEMPLATE.md`
- view: `product/views/PARKING_LOT_VIEW.md`
