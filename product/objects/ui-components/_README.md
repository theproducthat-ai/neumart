# UI Components

Stores objects representing reusable or route-specific UI components. These track where components live in the codebase, which screens use them, which features they belong to, and their current status.

## ID Format
`UIC-NNN`

## Linked Objects
- screens: `product/objects/screens/`
- features: `product/objects/features/`
- bugs: `product/objects/bugs/`

`product/os/templates/UI_COMPONENT_OBJECT_TEMPLATE.md`
