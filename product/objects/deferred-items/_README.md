# Deferred Items

Holds requests or ideas that were explicitly deferred to a future phase, sprint, or quarter. Unlike parking-lot items, deferred items have a target timeframe or trigger condition.

## ID Format
`DEF-NNN`

## Lifecycle
`deferred` → `backlog` | `cancelled`

## Linked Objects
- source requests in `product/objects/requests/`
- template: `product/os/templates/DEFERRED_ITEM_TEMPLATE.md`
- view: `product/views/DEFERRED_REQUESTS_VIEW.md`
