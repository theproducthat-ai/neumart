# Open Questions

Tracks questions that arise during product work and have not yet been resolved. Open questions block or constrain specific work items until answered. They must not be silently dropped.

## ID Format
`OPQ-NNN`

## Lifecycle
`open` → `answered` | `parked` | `obsolete`

## Linked Objects
- parent object (PRD, feature, story, roadmap item, etc.)
- discovery notes: `product/objects/discovery-notes/`

## Template
`product/os/templates/OPEN_QUESTION_TEMPLATE.md`

## Policy
`product/os/policies/MIDSTREAM_CHANGE_RULES.md`
