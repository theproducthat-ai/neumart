# QA Runs (Legacy V1)

**Object type**: `qa-run` (V1 format)  
**Owner**: QA Lead

## Note on This Folder

This folder is used for V1-style QA run artifacts. For V2 QA objects, use `product/objects/qa-tests/` instead.

## What Goes Here

- Detailed test case execution logs
- Test run scripts or automated test results
- QA artifacts that are too large for a single QA-NNNN object

## Relationship to qa-tests/

`qa-tests/` stores the QA test plan and summary object.  
`qa-runs/` stores the detailed execution artifacts linked from the qa-test object.
